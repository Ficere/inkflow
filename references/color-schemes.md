# inkflow Color Schemes

Six built-in themes. Pass the theme name as the last argument to `generate_images.py`.

## warm (default)
- Background: `rgb(248, 244, 236)` — warm off-white
- Text: `rgb(42, 36, 31)` — deep brown-black
- Accent: `rgb(66, 122, 118)` — muted teal
- **Best for:** Lifestyle, social commentary, general essays, emotional topics

## cool_blue
- Background: `rgb(235, 242, 250)` — cool light blue
- Text: `rgb(30, 36, 48)` — dark navy
- Accent: `rgb(52, 100, 168)` — medium blue
- **Best for:** Tech analysis, rational arguments, AI/science topics

## sage_green
- Background: `rgb(240, 245, 238)` — soft sage
- Text: `rgb(36, 42, 34)` — dark forest
- Accent: `rgb(76, 132, 88)` — leaf green
- **Best for:** Growth, wellness, environment, self-improvement

## dusty_rose
- Background: `rgb(248, 240, 240)` — muted blush
- Text: `rgb(48, 32, 36)` — deep rose-black
- Accent: `rgb(168, 82, 100)` — dusty rose
- **Best for:** Relationships, emotions, personal narratives, feminism topics

## amber
- Background: `rgb(250, 245, 232)` — warm amber-white
- Text: `rgb(48, 38, 24)` — dark brown
- Accent: `rgb(188, 132, 48)` — golden amber
- **Best for:** Economy, consumption, career, workplace topics

## slate
- Background: `rgb(238, 240, 244)` — blue-grey
- Text: `rgb(36, 38, 44)` — charcoal
- Accent: `rgb(88, 98, 128)` — slate blue
- **Best for:** Formal topics, workplace, institutional critique, serious analysis

---

## Selection Guide

| Article Topic | Recommended Theme |
|--------------|------------------|
| Social pain points (就业/消费/焦虑) | `warm` |
| Tech / AI / rational analysis | `cool_blue` |
| Personal growth / wellness | `sage_green` |
| Relationships / emotions | `dusty_rose` |
| Money / economy / career | `amber` |
| Workplace / institutional | `slate` |
