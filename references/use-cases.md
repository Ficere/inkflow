## 案例一：职场社交评论 → 公众号图文包

**输入文章**

```
假装上班的人，到底在假装什么？

每天早上九点，一群人浪潮般涌入商业区的咖啡店。他们开好电脑，打开招聘网站，然后开始刷简历。你以为他们是来面试的？不，他们是来偷时间的。

在一个不论论文还是码代码的空间里，他们以“自由职业者”的身份与世界保持对话。电脑前放着一杯现嫦式的南粗山拿铁，前方是反复刷新的职场动态。

“身份”是一件被穿上又脱下的外套。在家人面前你是“在苏州工作的小王”，在手机里你是“求职中的自由职业者”，在咖啡店里你是“在等 offer 的前”前端””。其实你只是在等一个还没有来的消息。

这种“假装上班”的状态，最大的伤害不是经济上的，而是让你逐渐失去了面对真实自己的勇气。当你不断用“忙”来掩饰资源的不足，你也在避开了真正需要做的决定。

解药不是“等到差不多了就好了”，而是“今天就告诉一个人真相”。无论是告诉父母你没有工作，还是告诉你自己你不想再偷时间了。背面的突破，往往比等待本身更有力量。
```

**输出配置**

| 参数 | 设定 |
|---|---|
| 标题 | 假装上班的人，到底在假装什么？ |
| 关键词 | 失业惨耻感, 身份绑架, 商业区咖啡店, 自由职业, 面对真实 |
| 配色 | `warm` |
| 封面图描述 | 一个人坐在空旷的办公楼咖啡区面前，电脑屏幕亮着，手边是冷却的咖啡杯，自然光线温暖 |
| 封面文案 | 失业后最难的，不是没有收入，而是你不知道该对谁告白 |
| 标签 | #职场 #成长 #社会观察 #自由职业 #心理 |
| 标题备选 | 十块钱的咖啡，是今天最大的交易成本｜假装上班的人，在假装什么？｜失业最大的消耗，不是钱 |

**生成结果**

- `01.png` 封面卡：温暖暖色调 + 标题 + 五个关键词引导
- `02–04.png` 正文页：分5段绒读，每页字号52px，行距1.65倍
- `publish_copy.md`：含推荐标题、封面文案、正文、标签、上传说明

## 案例二：科技评论 → 小红书图文模板

**输入文章**

```
大模型时代，普通人该怎么答对信息稠密的窄门？

当一个AI工具可以在两分钟内生成一份策划案，一个普通的文案从业者应该做什么？

答案不是抗拒，也不是撒手不干。答案是：去做AI做不了的事。

具体来说有三类事：一是“海量信息中的一条线”，二是“生活体验的猜想和试错”，三是“个人氛围的连续与一致”。

前者不是简单的“搜索”，而是在大量无用信息中找到那一条省略了但至关重要的线索。中者是在非正式场合中感受人们真实的情绪和偏好，而不是等到数据验证之后才行动。后者是最难的——让一个人看到文字时，能感觉到这就是“你”的创作，而不是“某个工具”的创作。

这三个方向其实对应着一种能力：向“内”发展的能力。AI放大了工具效率，但工具效率不会替代“你是谁”。
```

**输出配置**

| 参数 | 设定 |
|---|---|
| 标题 | 大模型时代，普通人该怎么答对？ |
| 关键词 | AI不能替代的, 信息稠密, 向内发展, 体验经验, 个人氛围 |
| 配色 | `cool_blue` |
| 封面图描述 | 未来感的蓝色冷光办公楼夜景，屏幕光映在人脸上，科幻风突如其来线窄门概念插画 |
| 封面文案 | AI 放大工具效率，但工具效率不会替代"你是谁" |
| 标签 | #AI #职场 #个人成长 #科技 #思维 |
| 标题备选 | 大模型时代，普通人该怎么答对？｜AI 时代的三个"向内"方向｜工具效率不会替代"你是谁" |

**生成结果**

- `01.png` 封面卡：冷蓝色调 + 科幻插画封面 + 标题
- `02–05.png` 正文页：4段落分页
- 发布文案包含小红书风格的短句和 emoji 建议

## 案例三：情感个人叙事 → 小绿书图文流

**输入文章**

```
我想离开上海了

我第一次来上海是五年前。那时候我对这座城市有很多幻想，觉得只要只够努力，结果都会有。

五年后我发现，结果确实会有，但是不一定是你想要的那种。

比如我有了更高的年薪，但也有了更差的睡眠。有了更好的房子，但也有了更少的朋友。有了更多的存款，但也有了更少的生活。

我常常在夜晚骑车经过回家的路。超过十二点的街道很安静，隔壁的建筑工地偶尔传来几声异响。我开始想，我真正想要的是什么。

也许不是离开，也许只是重新安排。不是放弃，而是选择。不是逃跑，而是换一个角度对待生活。
```

**输出配置**

| 参数 | 设定 |
|---|---|
| 标题 | 我想离开上海了 |
| 关键词 | 远离上海, 选择与放弃, 重新安排, 夜晚骑车, 答案不是逃跑 |
| 配色 | `dusty_rose` |
| 封面图描述 | 暗粉色调的夜晚城市街景，一个背影站在天桥上看远方，车流细线般的灯火从脚下流过，水彩插画风格 |
| 封面文案 | 五年后我发现，结果确实会有，但不一定是你想要的那种 |
| 标签 | #情感 #生活 #城市 #个人叙事 #心理 |
| 标题备选 | 我想离开上海了｜五年后，结果不是我想要的那种｜我们离开的到底是什么？ |

**生成结果**

- `01.png` 封面卡：暗粉色调 + 文艺插画 + 标题
- `02–06.png` 正文页：长文分段，沉浸式阅读
- 小绿书风格配图说明

## 案例四：批量生成 UP 主内容赋能包

**场景**

用户从某个知识类 UP 主处收集了 3 篇 Markdown 草稿，每篇包含「备选标题」「封面文案」「正文」「标签」，希望用 `generate_up_distill_cards.py` 批量转换为图文包。

**草稿目录结构**

```
drafts/
├── 01_ai_workflow.md
├── 02_investment_mindset.md
└── 03_urban_loneliness.md
```

**草稿格式示例**（`01_ai_workflow.md`）

```markdown
## 备选标题

AI 时代的三种工作流重构｜不是AI替代你，是AI放大你｜从工具到体系，普通人的升级路径

## 封面文案

当你把AI当工具用的时候，你只是看了个比例；当你把AI当体系构建的时候，你才看到了趋势。

## 正文

第一个误区是把AI当“搜索增强”......

## 标签

#AI工作流 #个人效率 #知识管理 #技术思维
```

**命令**

```bash
python scripts/generate_up_distill_cards.py \
  ./drafts \
  ./avatar.png \
  ./output \
  --zip ./output_bundle.zip
```

**生成结果**

```
output/
├── 01_ai_workflow/
│   ├── 01-封面.png
│   ├── 02-正文.png
│   └── 公众号发布文案.md
├── 02_investment_mindset/
│   ├── 01-封面.png
│   ├── 02-正文.png
│   └── 公众号发布文案.md
├── 03_urban_loneliness/
│   ├── 01-封面.png
│   ├── 02-正文.png
│   └── 公众号发布文案.md
├── 公众号发布文案合集.md
└── 生成清单.csv
```

**说明**

公开输出文件中不显示来源，发布口宫以“个人学习笔记”呈现。
